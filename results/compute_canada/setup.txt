Instructions for getting this repo working on a compute canada cluster

- if renv is displayed at startup, run renv::deactivate(), this will make life considerably easier
- module load gcc/9.3.0 r/4.3.1 gurobi/11.0.0
- install RcppEigen to personal directory
- install cvxr (requires RcppEigen) following repo instructions
- if prompted, tell it you want to update everything (option 1)

- install gurobi, use module show gurobi to find location of install
- provide this path to R for installing the corresponding R library
- install.packages('slam')
- modify path so that CVXR can talk to gurobi
- devtools::test(), make sure package runs (interactive)
- devtools::test(), run in a bash script
